# Icons Folder

Place your custom SVG icons from Figma here.

## File Naming Convention

- Use lowercase with hyphens: `icon-name.svg`
- Be descriptive: `home.svg`, `user-profile.svg`, `settings-gear.svg`
- Match the names in `components/shared/Icon/icons.tsx`

## Example

If you have `home.svg` in this folder, use it like:

```tsx
<Icon name="home" size={24} />
```
